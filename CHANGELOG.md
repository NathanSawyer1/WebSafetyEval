# Changelog

## v0.1.0

Initial MVP release of `web-safety-eval`.

Highlights:
- installable `web-safety-eval` CLI
- `run`, `run-all`, `list-scenarios`, `quickstart`, and `explain-results` commands
- OpenClaw CLI adapter as the primary real-agent path
- artifact-backed run outputs including reports, transcripts, and tool-call traces
- CLI-first README and setup flow
- focused regression coverage for CLI entrypoints and OpenClaw agent flag handling
